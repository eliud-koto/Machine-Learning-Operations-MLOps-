## Course Overview

**Lecturers:**  
- Oliver Angelil (Ishango.ai)  
- Cyrille Feudjio (Ishango.ai)  
- Vincent Frimpong (Ishango.ai)

This course covers essential **applied skills** to bridge the gap between **academia and industry** in 2025. It emphasizes best coding practices and industry standards for building **production-ready machine learning applications** that are deployed and hosted on **Google Cloud Platform (GCP)**.

---

## Topics Covered

- Best coding practices (e.g. **PEP8**, **CI/CD**, **type hinting**)  
- Technical debt  
- Microservice architectures and containerization  
- RESTful APIs  
- Basic User Interfaces (UIs)  
- Data Engineering  
- Distributed vs. single-node compute  
- The machine learning lifecycle  
- Agentic AI
